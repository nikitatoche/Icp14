Static position can refer to a posture or position that is held for a long time, or to a default position style in CSS. . The default position style for elements in a CSS layout . Elements with static positioning appear in the order they are defined in the HTML . Static positioning does not allow for adjustments to an element's position . Static positioning is a good choice for displaying standard content

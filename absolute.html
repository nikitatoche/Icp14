position: absolute; is a CSS property that allows elements to be positioned based on an absolute point on the page, relative to its parent element. This ensures the element's position is the same even if the page is resized or the content is changed

Absolute units specify a fixed length value. It doesn't matter if the screen's width or height changes, the value will remain fixed. For high-resolution media like print documents, it is recommended that you use cm , mm , or pt . For webpages, px is the recommended unit.
